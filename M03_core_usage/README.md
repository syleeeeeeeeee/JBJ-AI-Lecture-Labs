# M03 · 핵심 사용법

## 실습 목표

VS Code에서 에이전트를 실행하고 맥락, 권한, 세션, 슬래시 명령을 다룹니다.

## 실습 순서

1. 새 연습 폴더를 VS Code로 엽니다.
2. 통합 터미널에서 Claude Code 또는 Codex를 실행합니다.
3. `/init`으로 `CLAUDE.md` 또는 `AGENTS.md` 초안을 생성합니다.
4. 다음 요청을 차례로 실행하고 에이전트가 탐색한 파일과 실행한 명령을 확인합니다.

```text
이 폴더에 어떤 파일들이 있는지 알려줘.
README.md를 만들고 이 프로젝트의 목적을 세 문장으로 정리해줘.
README.md에 완료 체크박스 한 개를 추가해줘.
```

5. 파일 수정 승인을 한 번 직접 검토합니다.
6. 세션을 종료한 뒤 최근 세션을 재개해 앞선 작업 내용을 물어봅니다.

## 완료 확인

- [ ] 프로젝트 폴더 안에서 에이전트를 실행했다.
- [ ] `/init` 결과를 확인했다.
- [ ] 파일 생성 또는 수정 작업을 완료했다.
- [ ] 승인 범위와 현재 작업 폴더를 확인했다.
- [ ] 종료한 세션을 재개했다
