# M15 · 생성형 미디어 종합

## 실습 목표

생성형 이미지와 짧은 영상 클립을 M14의 Remotion 프로젝트에 결합해 15초 완성 영상을 만듭니다.

## 소스 준비

- 직접 생성한 16:9 배경 이미지 1~2장
- 직접 생성한 짧은 영상 클립 1개
- 영상에 표시할 제목과 자막
- 출처와 사용 권한을 확인한 폰트·음원만 사용

## 실습

```text
assets 폴더의 이미지와 영상 클립을 Remotion으로 엮어 15초 소개 영상을 만들어줘.
처음에는 제목, 중간에는 클립과 자막, 마지막에는 마무리 문구가 나오게 해줘.
이미지 비율을 왜곡하지 말고 영상과 자막이 겹치지 않게 배치해줘.
Studio에서 검토한 뒤 MP4로 렌더할 수 있게 해줘.
```

완성본에서 화면 비율, 자막 가독성, 소스 전환, 음량, 라이선스를 점검합니다.

## 완료 확인

- [ ] 생성 이미지와 영상 클립을 준비했다.
- [ ] Remotion에서 소스를 결합했다.
- [ ] 제목과 자막이 안전 영역 안에 표시된다.
- [ ] 15초 MP4 렌더링을 완료했다.
- [ ] 사용한 소스의 권리와 출처를 확인했다.
