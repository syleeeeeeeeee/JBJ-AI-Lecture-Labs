# M3_usage · 핵심 사용법 — 스킬 구현 실습 자료

## 실습 목적
영수증 사진을 폴더에 넣으면 AI가 **카테고리별 폴더로 정리**하고 **회계장부 엑셀을 자동 작성**하는 스킬(Skill)을 직접 만들어 봅니다.
비정형 데이터(영수증 이미지) → 정형 데이터(회계장부 엑셀) 변환을 에이전트가 대신하는 실무형 자동화 실습입니다.

## 파일 안내

| 파일 | 용도 |
|---|---|
| `회계장부_실습.xlsx` | 영수증 정리 결과를 기록할 엑셀 양식 (3시트: 영수증입력 · 회계장부 · 카테고리별집계) |
| `스킬 학습용.zip` | 스킬 제작·학습용 영수증 10장 (receipt_ko_01~10) |
| `스킬 테스트용.zip` | 완성한 스킬을 검증할 영수증 10장 (receipt_ko_11~20) |

## 실습 순서
1. `스킬 학습용.zip` 압축을 풀어 작업 폴더에 넣습니다.
2. 에이전트에게 요청합니다: *"이 영수증들을 카테고리별로 정리하고 회계장부 엑셀로 작성하는 스킬을 만들어줘."*
3. 스킬 실행 후 `정리완료/<카테고리>/` 폴더와 결과 엑셀 파일을 확인합니다.
4. `스킬 테스트용.zip`으로 재실행해 처음 보는 영수증에도 분류가 정확한지 점검합니다.
