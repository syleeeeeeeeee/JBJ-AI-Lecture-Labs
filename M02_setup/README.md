# M02 · 환경 구축 · 설치

## 실습 목표

Markdown과 JSON의 용도를 이해하고 AI 에이전트 실행 환경을 점검합니다.

## 환경 점검

VS Code의 통합 터미널에서 실행합니다.

```powershell
node --version
npm --version
git --version
code --version
claude --version
codex --version
```

수업에서 선택한 도구 하나만 정상 실행되어도 됩니다. 설치 명령과 인증 방식은 각 도구의 최신 공식 안내를 따릅니다.

## Markdown 실습

`practice.md`를 만들고 제목, 목록, 체크박스, 코드 한 줄을 작성한 뒤 VS Code 미리보기로 확인합니다.

```markdown
# 나의 첫 프로젝트

- 목표: AI 에이전트 익히기
- [ ] 환경 점검
- [ ] 첫 파일 만들기

`node --version`
```
