# JBJ-AI-Lecture-Labs

Notion의 **AI Agent 입문 과정 M1~M15**에 맞춘 학생 배포용 실습 저장소입니다.
강의 내용과 슬라이드는 Notion에서 확인하고, 이 저장소에서는 실습 절차와 필요한 파일만 사용합니다.

## 시작 전 준비

- Windows PC 또는 macOS
- VS Code, Node.js 18 이상, Git
- Claude Code 또는 Codex
- GitHub 계정
