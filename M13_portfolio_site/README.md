# M13 · 개인 소개 홈페이지 만들기

## 실습 목표

단일 HTML 소개 페이지를 만들고 GitHub Pages로 배포·수정합니다.

## 제작

새 저장소 또는 별도 프로젝트 폴더에서 에이전트에게 요청합니다.

```text
나의 개인 소개 홈페이지를 index.html 한 파일로 만들어줘.
이름, 한 줄 소개, 관심 분야, 프로젝트, 연락 링크를 포함하고 모바일에서도 읽기 쉽게 만들어줘.
외부 빌드 도구 없이 브라우저에서 바로 열려야 해.
```

`index.html`을 브라우저로 열어 내용, 링크, 모바일 폭을 확인합니다.

## 배포

1. GitHub 저장소에 `index.html`을 Commit하고 Push합니다.
2. 저장소 **Settings → Pages**에서 배포할 브랜치와 루트 폴더를 선택합니다.
3. 발급된 URL을 열어 확인합니다.
4. 문구나 색상을 한 번 수정해 다시 Push하고 배포 갱신을 확인합니다.

## 완료 확인

- [ ] 단일 `index.html` 페이지를 만들었다.
- [ ] 데스크톱과 모바일 폭에서 확인했다.
- [ ] GitHub Pages URL로 접속했다.
- [ ] 수정 후 재배포했다.
